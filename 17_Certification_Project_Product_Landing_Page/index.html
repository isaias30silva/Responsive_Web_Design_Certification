How to choose?

When choosing a computer, you can consider things like:

  • Processor: The processor is like the brain of the computer, and works with the system memory to determine how fast programs run, how many programs can be open at once, and the complexity of software that can be run.
  • RAM: RAM stands for Random Access Memory, and is like the computer's short-term memory. RAM works with the processor to help the computer run efficiently.
  • Storage: Storage is where files are saved locally on the computer, and also holds the operating system. If you have multiple large files, you may need a larger storage device.
